Migration runner & MySQL wrapper for https://github.com/testcontainers/testcontainers-go
## Development
1. Make sure Docker is installed on your local machine
   ```
   docker --version
   ```
2. Make sure you can run Docker locally
   ```
   docker run hello-world
   ```
3. Run the test
   ```
   ./gotest.sh
   ```
